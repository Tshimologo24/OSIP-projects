# Capture Network traffic using Wireshark

Objective is capturing and analyzing network traffic using Wireshark. 

Tools used is Wireshark 

# Steps taken: 
* Installed Wireshark. 
* started capturing traffic on my local network.
* Applied http filter traffic and
* Analyzed the packets

# What I have analyzed:
* Packets capyured show no suspicious activity
* A standard GET request was sent to the server
* Systems checking for updates
* 304 not modified show efficient caching
